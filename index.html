"The Extra Step" Mobile Application

"The Extra Step" is an innovative mobile application designed to promote sustainable transportation choices. Broadband Speed Prediction with Scikit-learn

This advanced machine learning project showcases my expertise in data science and predictive modeling. Utilizing Scikit-learn, I developed a sophisticated model to predict broadband plan speeds based on various factors such as service provider, connection type, and price. The project involved extensive data preprocessing, feature engineering, and the application of multiple machine learning algorithms. I employed GridCV for hyperparameter tuning, ensuring optimal model performance. The final model was selected based on its superior accuracy and reliability in predicting broadband speeds. To make this tool accessible and user-friendly, I deployed it on a Streamlit-hosted website, allowing users to input their parameters and receive instant predictions. This project not only demonstrates my proficiency in machine learning techniques but also my ability to create practical, user-oriented solutions for real-world problems in the telecommunications industry.

PowerGrapher: Automated Review Analysis and Visualization

"PowerGrapher" represents a pinnacle in my Robotic Process Automation (RPA) expertise, showcasing the power of automation in data analysis and visualization. Developed using UiPath, this sophisticated tool automates the entire process of gathering and analyzing Google Maps reviews. It begins by scraping review data, including content, ratings, author information, and likes. This data is then meticulously compiled into an Excel sheet for structured storage. The true innovation lies in the automatic generation of an HTML file containing interactive, visually appealing graphs that represent the collected data. To add depth to the analysis, I integrated the Google Gemini API, which generates AI-powered summaries of the reviews, distilling key insights without human intervention. This project not only demonstrates my proficiency in RPA and data visualization but also showcases my ability to create end-to-end automated solutions that transform raw data into actionable insights, all without manual input.
